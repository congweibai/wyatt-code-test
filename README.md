## Prepare

Node Version: v20.16.0

## Commands

| name                 | commands           |
| -------------------- | ------------------ |
| install dependencies | `npm install`      |
| run locally          | `npm run dev`      |
| run test             | `npm run test`     |
| test coverage        | `npm run coverage` |
| prepare              | `npm run prepare`  |
